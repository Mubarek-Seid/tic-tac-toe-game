#  Tic Tac Toe game

A simple Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript** while practicing the **Module Pattern (IIFE)** and **Factory Functions**.

## Features

- Two-player mode (Player 1 vs Player 2)
- Turn switching between X and O
- Winner detection
- Draw detection
- Reset game functionality
- Dynamic board rendering
- Modular JavaScript structure

## Technologies Used

- HTML
- CSS
- JavaScript

## Concepts Practiced

This project helped me practice:

- Module Pattern (IIFE)
- Factory Functions
- DOM Manipulation
- Event Listeners
- Game Logic
- Array Handling
- State Management
- Separation of Concerns

## Project Structure

- **Gameboard Module**
  - Stores and updates board data

- **GameController Module**
  - Handles turns
  - Checks winners and draws
  - Controls game flow

- **DisplayController Module**
  - Updates UI
  - Handles user clicks
  - Displays winner

## How to Play

1. Open `index.html`
2. Click an empty square
3. Players take turns placing X and O
4. First player to make three in a row wins
5. Press **Reset** to start a new game
